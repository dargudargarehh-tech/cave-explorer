# The Cave Explorer A fast 2D twin-stick action-platformer in vanilla JS. Survive procedurally generated caves, defeat bosses, and get loot!  **Controls:** WASD/Arrows (Move), Space (Jump), Mouse (Aim), Click/F (Shoot). **Play:** Open `https://cave-explorer-ten.vercel.app` in your browser!
